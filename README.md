# Booklib API

A simple book review application.

## Stack

- Maven
- Spring Boot
- Spring JPA
- H2 Database
- Liquibase
- SpringDoc OpenAPI
- Mapstruct
- Lombok
- jUnit

## Getting started

To run application:

```sh
mvn spring-boot:run
```

To run tests:

```sh

mvn test
```

API documentation: http://localhost:8080/swagger-ui/index.html
